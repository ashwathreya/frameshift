# Frameshift Artist Flow Prototype

This folder contains the uploaded artist dashboard prototype, split into separate HTML, CSS, and JavaScript files so it can be evolved into a production web app.

- `index.html` contains the page structure.
- `styles.css` contains design tokens, layout, and component styling.
- `app.js` contains the prototype state and UI behavior.

Open `index.html` directly in a browser for now. When the app stack is chosen, this can be migrated into route/page components, reusable UI components, and API-backed data stores.
